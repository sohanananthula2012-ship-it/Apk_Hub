# Contributing to APK Hub

Thank you for contributing to APK Hub, a community-driven Android marketplace built by GPUC.

## Ways to Contribute
- Frontend (React, TypeScript, UI/UX)
- Backend (Node.js, Express, APIs)
- Android SDK (Kotlin)
- Security
- DevOps
- Documentation
- Testing and QA

## Recognition
Every contributor will:
- Be listed on the APK Hub Hall of Fame.
- Be credited on the project landing page.
- Be able to showcase their work in their portfolio and résumé.
